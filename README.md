# Tic-Tac-Toe_Game
🎮 Tic-Tac-Toe Game is a fun and interactive web-based game built using HTML, CSS, and JavaScript. Players take turns marking X and O on a 3×3 grid, aiming to get three symbols in a row. The project features winner detection, draw handling, responsive design, smooth gameplay, and a clean user-friendly interface. 🚀
